Global Earthquake Data Visualization and Analysis

This project utilizes Microsoft Fabric to develop a comprehensive data pipeline and interactive visualization for global earthquake data sourced from the USGS API. The solution focuses on real-time data ingestion, processing, and visualization to provide insights into worldwide seismic activity.

Data Source

Earthquake API Data Source: API Website

Key Features

  • Data Pipeline: Automated ingestion and transformation of USGS earthquake data using Microsoft Fabric's Lakehouse, Data Factory, and Spark Notebooks.
  • Interactive Visualization: A Power BI report featuring a global map displaying earthquake locations, significance levels, and interactive filtering.
  • Significance Classification: Earthquakes are categorized as "High", "Moderate" or "Low" based on their significance.
  • Date Range Filtering: Users can analyze earthquake data within specific timeframes.
  • Location Enrichment: Data is enriched with country codes for enhanced geographical context.

Technologies Used

  • Microsoft Fabric (Lakehouse, Data Factory Pipelines, Notebooks, Environments)
  • PySpark
  • Python
  • USGS API
  • Power BI
  • reverse_geocoder
